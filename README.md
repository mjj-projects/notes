## 简介

本项目属于个人学习项目，主要用于梳理学习流程，记录学习内容与学习笔记。

由于本人是一名 Java 语言开发者，需要学习的内容很多，技术栈的要求也很深。因此，为了提高自己的学习效率，改变自己的学习方式，方便自己以后的复习，最终创建了这个项目。

## 简介

本项目是一个学习项目，用于记录本人在学习过程中的笔记。本项目主要将「未完全理解」或「存疑」的知识点进行记录与分析，以巩固对其知识点的理解和掌握。

> 如何验证理解和掌握程度？

本人通过「笔记」来验证自己的理解程度，通过「示例」来验证自己的掌握程度。

—— **个人观点：完全理解，可以记录下来或讲给别人听等等；完全掌握，需要不断的实践和试错。**
